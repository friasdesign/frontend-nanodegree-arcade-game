setCollider

(
  • confObj
)

Defined in js/app.js:669

This method sets the collider's properties based on values passed in a configuration object. Here is an example of usage, this example since no value has been passed to originX, it uses the default value:

this.collider.setCollider({
  originY: grid.boxHeight(),
  height: grid.boxHeight() * 0.7,
  width: grid.boxWidth() * 0.85
});

Parameters:

  • confObj Object

    The configuration object. It holds the properties: originX, originY, width, height. These properties are optional and default to the current value for this.
